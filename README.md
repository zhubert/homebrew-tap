# homebrew-tap

Custom Homebrew tap for personal tools.

## Installation

```bash
brew tap zhubert/tap
```

## Formulas

### plural

A TUI tool for managing multiple concurrent Claude Code sessions in isolated Git worktrees.

```bash
brew install --HEAD zhubert/tap/plural
```

**Requirements:**
- Claude Code CLI (authenticated)
- Git
- GitHub CLI (optional, for PR functionality)

**Usage:** Run `plural` to launch the TUI.

**Source:** https://github.com/zhubert/plural
